# MCU+ SDK DMAUTILS

</div>

## Introduction

MCU+ SDK DMAUTILS is an auxiliary repository used along with [MCU+ SDK](https://github.com/TexasInstruments/mcupsdk-core-k3) 
This repo mainly contains the DMA Utils library required for AM62A C7x

## Usage in MCU+ SDK

This repo is meant to be used with MCU+ SDK repo.
Refer [MCU+ SDK Core repository](https://github.com/TexasInstruments/mcupsdk-core-k3) for 
more details regarding the MCU+ SDK.
